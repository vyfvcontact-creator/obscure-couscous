# 3/13-AI Engine v1.0

نظام ذكي يعمل كخادم (Server) داخل بيئة Termux، مرتبط عبر مكتبة Baileys بواتساب.

## الهدف الأساسي
تقديم حلول أكاديمية دقيقة (رياضيات، فيزياء، علوم، وجميع المواد الدراسية) بسرعة عالية، مع استهلاك منخفض للموارد، والعمل بشكل مستقر 24/7.

## التثبيت

1. استنسخ المشروع
2. قم بتثبيت التبعيات: `npm install`
3. قم بتحديث `config.js` بمفاتيح API لـ Gemini و Groq ومعرف الإدارة
4. شغل البوت: `npm start` أو استخدم PM2 للإدارة المستمرة

## التشغيل في Termux (الإصدار العادي)

البوت متوافق مع Termux العادي من Google Play Store أو التحميل المباشر للـ APK الرسمي.

### الطريقة السريعة (موصى بها):
```
bash setup-termux.sh
```
ثم اتبع التعليمات على الشاشة.

### الخطوات التفصيلية:

1. **تثبيت Termux:**
   - قم بتحميل Termux من Google Play Store أو التحميل المباشر للـ APK الرسمي
   - افتح Termux وانتظر التحديث الأولي

2. **تحديث النظام:**
   ```
   pkg update && pkg upgrade
   ```

3. **تثبيت Node.js:**
   ```
   pkg install nodejs
   node --version  # تأكد من الإصدار 18+
   ```

4. **تثبيت Git:**
   ```
   pkg install git
   ```

5. **استنساخ المشروع:**
   ```
   git clone <repository-url>
   cd <project-folder>
   ```

6. **تثبيت التبعيات:**
   ```
   npm install
   ```

7. **تحديث التكوين:**
   - افتح `config.js` بمحرر نصوص
   - أضف مفاتيح API الحقيقية:
     ```javascript
     geminiApiKey: 'your-actual-gemini-key',
     groqApiKey: 'your-actual-groq-key',
     adminId: '1234567890@s.whatsapp.net'  // رقمك
     ```

8. **تشغيل البوت لأول مرة:**
   ```
   npm start
   ```
   - سيظهر رمز QR - امسحه بتطبيق WhatsApp
   - انتظر رسالة "Connected to WhatsApp"

9. **إيقاف البوت (Ctrl+C) وتشغيله مع PM2:**
   ```
   pm2 start ecosystem.config.js
   ```

10. **التحقق من التشغيل:**
    ```
    pm2 status
    pm2 logs 3-13-ai-engine
    ```

### أوامر PM2 المفيدة:
- `pm2 restart 3-13-ai-engine` - إعادة التشغيل
- `pm2 stop 3-13-ai-engine` - إيقاف
- `pm2 delete 3-13-ai-engine` - حذف
- `pm2 monit` - مراقبة في الوقت الفعلي

### استكشاف الأخطاء:
- إذا لم يتصل: تأكد من اتصال الإنترنت وحاول إعادة المسح
- إذا توقف: `pm2 restart` أو `npm start` مرة أخرى
- للسجلات: `pm2 logs` أو تحقق من مجلد `logs/`

### نصائح للاستقرار:
- لا تغلق Termux أثناء التشغيل
- استخدم `pm2 startup` للتشغيل التلقائي عند إعادة التشغيل
- احتفظ بنسخة احتياطية من `session.json` لتجنب إعادة المسح

## الاستخدام

- أرسل صورة للتمارين أو الامتحانات للحل التلقائي الفوري باستخدام Gemini
- أرسل نص مباشر في الدردشات الخاصة أو منشن البوت في المجموعات للحصول على إجابات منظمة ومختصرة باستخدام Groq
- استخدم الأوامر الأكاديمية: !solve, !explain, !resume للاستجابات السريعة باستخدام Groq
- الأوامر الترفيهية: !game, !joke, !quote, !fact
- الأوامر الإدارية: !logs, !status, !restart (للإدارة فقط)

## تنسيق الإجابات

جميع الإجابات تكون:
- **دقيقة ومنظمة** مع خطوات واضحة
- **مختصرة** مع التركيز على المحتوى الأساسي
- **منسقة** باستخدام Markdown للقوائم والعناوين
- **باللغة العربية** مع إمكانية الإنجليزية حسب الطلب

## محركات الذكاء الاصطناعي

- **Gemini**: لمعالجة الصور وحل الامتحانات من الصور
- **Groq**: للاستجابات البسيطة والسريعة في جميع المواد الدراسية

## الهيكل

- `/commands` - معالجات الأوامر
- `/services` - محركات الذكاء الاصطناعي (Text: Groq, Vision: Gemini)
- `/core` - النواة الأساسية
- `/db` - قاعدة البيانات للذاكرة

## ميزات المجموعات

البوت يعمل بشكل مثالي في المجموعات مع:
- الرد فقط عند المنشن (@bot) أو الأوامر المباشرة
- منع السبام بحد أقصى 5 ثوانٍ بين الأوامر
- رسائل ترحيب للأعضاء الجدد
- أمر !groupinfo لعرض معلومات المجموعة
